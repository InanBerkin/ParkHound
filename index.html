Parkhound

Senior Design Project

ParkHound is a cross-platform application that will help users to find available parking spots in an outdoor parking lot. Currently, this can be done with sensors in an indoor parking lot. ParkHound will provide a similar service without having to install sensors near each of the parking spots using object detection and share this information via an application, so users can see the available parking spots without having to drive around the parking lot.
